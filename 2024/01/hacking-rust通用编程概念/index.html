变量和可变性

默认情况下，变量是不可变的。以下代码是错误的：

1
2
3
4
5
6

fn main() {
    let x = 5;
    println!("The value of x is: {}", x);
    x = 6;
    println!("The value of x is: {}", x);
}

如果需要更改变量，需要声明可变类型：

1

let mut x = 1;

常量

常量是绑定到一个常量名且不允许更改值。常量和变量之间存在一些差异：

  • 常量无法使用 mut，常量自始自终都无法改变，其语法如下：

    1

    const THREE_HOURS_IN_SECONDS: u32 = 60 * 60 * 3;

  • 常量的值必须注明类型。

  • 常量可以在任何作用域内声明，包括全局作用域。

  • 常量一般用大写字母加下划线来命名。

常量表达式常常用计算式这种更容易理解的方式来命名，而不是直接计算出结果。

遮蔽

我们可以声明和前面变量相同名称的变量，但是会覆盖原来变量的值。

1
2
3

let x = 1;
let x = x + 1;
// 结果是：x = 2

mut 和遮蔽之间的区别：遮蔽可以改变变量的类型，而 mut 不可以。

数据类型

Rust 是强数据类型的语言，也是静态类型语言，意味着它必须在编译期知道所有变量的类型。编译器通常可以根据值和使用方式推断出数据类型，但是一旦类型可能是多种情况的时候，就必须要加上一个类型标注，类型标注如下：

1

let x: u32 = 89;

标量类型

整型

整型 有符号类型 无符号类型
8 位 i8 u8
16 位 i16 u16
32 位 i32 u32
64 位 i64 u64
128 位 i128 u128
arch isize usize

整型的默认类型是 i32。

其中，isize 和 usize 类型取决于计算机体系结构：若是 64 位架构则是 64 位，32 位架构则为 32 位。

Tips：在表示数字字面量的时候，可以使用 _ 作为分隔符方便读数，如 98_222。

整型溢出：当数据超出数据类型能够表示的范围的时候，就会发生整型溢出。当使用 dubug 模式编译时，Rust 会检查整型溢出，如果发现了就会报出 panic 的错误。但是使用 release 模式编译的时候，不检测会导致 panic 的整型溢出，而是会对溢出的数据采取一种二进制包裹的操作。大于该类型最大值的数值会被“包裹”成该类型能够支持的对应数字的最小值。比如在 u8 的情况下，256 变成 0，257 变成 1。

浮点型

Rust 中浮点型分为 f32 和 f64，默认类型是 f64。

数字运算

Rust 支持所有的数学运算。整数除法会向下取整。

bool 类型

1

let f: bool = false;

在 Rust 中，数字是不能替代布尔值进行条件判断的。以下这段代码会报错：

1
2
3
4

let y = 0;
if y {
    println!("hello");
}

字符类型

char 类型用单引号括起来，而字符串采用双引号括起来。

Rust 字符类型大小是 4 个字节，表示的是一个 Unicode 标量值。

1
2

let c = 'z';
let heart_eyed_cat = '😻'

复合类型（将多个标量类型组合成一个类型）

元组类型

元组长度固定，一旦定义后就不能更改长度。元组中的数据类型不要求是相同的。声明方法如下：

1
2
3

fn main() {
    let tup = (200, 8.4, 1);
}

也可以自定义每个元素的数据类型：

1
2
3

fn main() {
    let tup: (i32, f64, u8) = (500, 6.4, 1);
}

获取元组中的值可以采用模式结构(destructure)的方式和句点的方式索引元素：

1
2
3
4
5
6
7

fn main() {
    let tup = (200, 8.4, 1);
    // 使用模式结构获取元素
    let (x, y, z) = tup;
    // 使用句点索引第一个元素 200
    let dot = tup.0
}

没有任何元素的元组是一种特殊类型，也只有一个值，被称为单元类型，该值被称为单元值。如果表达式不返回任何其他值，就隐式地返回单元值。

数组类型

与元组的区别：数组的每个元素必须具有相同的类型。Rust 中的数组也具有固定长度。因此数组没有 vector（向量，动态数组）那么灵活。

Tips：如果不确定要使用数组还是 vector 的话，那就应该使用 vector。

1
2
3
4
5

fn main() {
    // 数组的声明方式
    // 一般声明
    let a = [1, 2, 3, 4, 5];
}

需要注意的是，因为数组的每个元素类型必须相同，因此无法单独指定每个元素的数据类型。声明格式接受两个参数，所有元素的类型和元素的个数，参数之间要用分号隔开：

1
2
3
4

fn main() {
    // i32 表示所有元素是 i32 类型，5 表示一共有 5 个元素
    let arr: [i32; 5] = [1, 2, 3, 4, 5];
}

获取数组元素的方式也和元祖不一样。数组元素的获取用的是 [] 的索引方式：

1
2
3
4
5

fn main() {
    let arr = [1, 2, 3];
    // 获取数组第一个元素
    let first_value = arr[0];
}

如果索引大于或等于数组长度，Rust 就会出现 panic。

函数

main() 函数是程序的入口，fn 关键字用来声明新函数。

Rust 函数和变量名使用下划线命名法，所有字母都是小写并且采用下划线分隔。

1
2
3
4
5
6

fn main() {
    print_hello_world();
}
fn print_hello_world() {
    println!("hello world");
}

Rust 并不在乎函数定义的先后顺序，如果函数定义在调用之后，不需要像 C 语言一样在前面声名函数原型。

参数

Rust 中定义的函数需要显示指定参数的数据类型，多个参数之间用逗号隔开，如下代码所示：

1
2
3
4
5
6
7

fn main() {
    another_function(32, 4.3);
}

fn another_function(x: i32, y: f64) {
    println!("The value of x and y is : {} {}", x, y);
}

语句和表达式

语句是执行一些操作但是不返回值的指令，而表达式计算并返回一个值。比如 let 关键字创建变量并绑定到一个值是一个语句。

语句不返回值，因此不能把语句赋给另一个变量。

表达式会计算一个值并将值返回。比如，我们用来创建新作用域的大括号（代码块）是一个表达式。

1
2
3
4
5
6

fn main() {
    let y = {
        let x = 3;
        x + 1
    };
}

带有返回值的函数

函数可以向调用它的代码返回值，我们并不对返回值命名，但是要在箭头（->）后声明它的类型。大部分函数隐式地返回函数体最后一个表达式的值。我们也可以使用 return 关键字提前返回指定值。

1
2
3
4
5
6
7
8

fn five() -> i32 {
    5
}

fn main() {
    let x = five();
    println!("the value of x is {}", x);
}

值得注意的是，一旦给表达式加上分号，表达式就变成了语句。比如下面这段实例代码：

1
2
3
4
5
6
7
8

fn main() {
    let x = five(1);
    println!("x if {}", {});
}

fn five(x: i32) -> i32 {
    x + 1
}

此时运行结果是正确的，返回值为 2。但是一旦在 x + 1 表达式后面加上分号，就变成了一个语句，无返回值，程序会报错。

在 let 语句中使用 if

在 let 语句中使用 if 表达式的时候，需要注意表达式前后值类型必须相同，否则会报错。

1
2
3
4
5
6

fn main() {
    let condition = true;
    let number = if condition { 5 } else { 6 };

    println!("The value of number is: {}", number);
}

循环

Rust 有 3 种循环结构：loop, while, for。

loop

loop 关键字告诉 Rust 一遍又一遍地执行代码直到手动结束为止，也可以添加 break 语句终止循环。

continue 关键字告诉程序跳过本次循环迭代中的任何代码，并转到下一次循环中。

Rust 还支持循环标签的功能，参见下面这段代码：

fn main() {
    let mut count = 0;
    'counting_up: loop {
        println!("count = {}", count);
        let mut remaining = 10;

        loop {
            println!{"remaining = {}", remaining};
            if remaining == 9 {
                break;
            }
            if count == 2 {
                break 'counting_up;
            }
            remaining -= 1;
        }
        count += 1;
    } 
    println!("End count = {}", count);
}

在外层循环中定义了一个标签 counting_up，这样就可以在内部循环中终止外部循环。注意循环标签的格式是一个单引号后面跟标签名称。
